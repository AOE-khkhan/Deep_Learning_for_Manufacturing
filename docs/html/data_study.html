Data and Convergence StudyÂ¶

The data study file is used to study the training data requirement for convergence, the file trains the model incrementally (training size is increased) based on the parameters specified in the modelconfig_train.py file and simultaneously tests the trained model at each step to estimate the optimum number of training samples required. The file also generates combined html based plots (using plotly and cufflinks), trained model, accuracy metrics and training plots for each iteration within the file structure
